Directions for opening Catalog

 1. Navigate to vagrant/catalog in commandline
 2. Run vagrant (vagrant up and then vagrant ssh)
 3. Navigate back to /vagrant/catalog
 4. Run database_setup.py (python database_setup.py) in order to create the catalog database
 5. Run lotsofcategories.py (python lotsofcategories.py) in order to populate the catalog database
 6. Run application.py (python application.py) to run the application
 7. Navigate to http://localhost:8000/ on your computer/laptop to access the homepage
